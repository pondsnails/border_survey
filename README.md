# 東京科学大学（旧東工大）系所属ボーダー調査ツール
このリポジトリは、東京科学大学（旧東京工業大学）の理工系学生向け「系所属ボーダー点」の調査・集計作業をカンタンかつ正確に行うための自動化ツールセットです。
毎年SNS等で行われる有志の調査を、Googleフォームの自動生成からデータ集計まで自動化し、次代の後輩たちへスムーズに引き継げるように作成しました。
## 📦 含まれるファイル
- `create_form_script.js`: アンケート用のGoogleフォームをワンクリックで自動生成するGAS（Google Apps Script）。
- `analyze_survey.py`: 回答データ（CSV）を読み込み、自動でボーダー点を算出して表を出力するPythonスクリプト。
---
## 🚀 使い方
### 1. アンケートフォームの作成 (Google Apps Script)
手作業でアンケートを作る必要はありません。以下の手順で一瞬で作成できます。
1. **Googleドライブ**を開き、「新規」>「その他」>「Google Apps Script」を選択します。
2. 開いたエディタにあるデフォルトのコードを消し、本リポジトリの `create_form_script.js` の内容をすべて貼り付けます。
3. エディタ上部のプロジェクト名を「系所属アンケート作成スクリプト」などに変更します。
4. ツールバーのプルダウンメニューから `createAffiliationSurveyForm` 関数が選択されていることを確認し、「**実行**」をクリックします。
   - ※初回のみGoogleからのシステム権限承認が求められます。「権限を確認」>ご自身のアカウント選択>「詳細」>「...（安全ではないページ）に移動」>「許可」と進んでください。
5. 画面下部の実行ログに `Published URL` が表示されます。これが**アンケートの回答用URL**です。SNS等でシェアして回答を集めましょう。
### 2. 回答データの収集
ある程度回答が集まったら、データをダウンロードします。
1. 作成されたGoogleフォームの「編集画面」を開きます。
2. 上部の「**回答**」タブを開き、「**スプレッドシートにリンク**」をクリックします。
3. 作成されたスプレッドシートを開き、メニューの `ファイル` ＞ `ダウンロード` ＞ `カンマ区切り形式 (.csv)` を選択します。
4. ダウンロードしたファイルの名前を **`survey_data.csv`** に変更し、`analyze_survey.py` と同じフォルダに配置します。
### 3. ボーダーの自動集計 (Python)
Pythonを使ってデータを集計し、瞬時にボーダー表を出力します。
※Pythonと `pandas` ライブラリがインストールされている必要があります（`pip install pandas`）。
コマンドライン（ターミナル等）でスクリプトを実行します。
```bash
python analyze_survey.py
```
実行が完了すると、同じフォルダに以下の2つのファイルが出力されます。
- **`border_analysis.csv`**: 全17系の「学院内/外」の合格・不合格ボーダー点の一覧データです。Excel等で開いて表や画像として公開できます。
- **`comments.txt`**: 回答者が書いてくれた「感想・後輩へのアドバイス」がテキスト形式でまとまっています。Note記事やブログ執筆時の素材としてコピペして使えます。
---
## 📝 貢献・カスタマイズ
年度が変わって系の名前やルール（例えば志望可能な最大数など）が変わった場合は、`analyze_survey.py` 内の `DEPT_SCHOOL_MAP` や `create_form_script.js` の項目名を直接書き換えるだけで対応可能です。
